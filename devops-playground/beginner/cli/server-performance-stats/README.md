# Server Performance Stats

## Overview
Build a command-line tool to analyze and display server performance statistics.

## Skill Level
Beginner

## Domain
CLI / System Monitoring

## Objectives
- Learn to read system metrics from `/proc` filesystem
- Parse and format system information
- Create user-friendly CLI output
- Understand CPU, memory, and disk metrics

## Project Requirements
Create a script that displays:
- CPU usage (overall and per core)
- Memory usage (total, used, free)
- Disk usage
- Top 5 processes by CPU usage
- Top 5 processes by memory usage

## Suggested Technologies
- Bash/Python/Go
- System utilities: `top`, `ps`, `df`, `free`

## Learning Outcomes
- Understanding Linux system metrics
- CLI tool development
- Data parsing and formatting
- Process monitoring

## Next Steps
After completing this project, try:
- Adding network statistics
- Creating historical tracking
- Implementing alerting thresholds
