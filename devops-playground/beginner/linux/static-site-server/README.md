# Static Site Server

## Overview
Set up a web server to host a static website.

## Skill Level
Beginner

## Domain
Linux / Web Servers

## Objectives
- Install and configure a web server
- Understand HTTP and web hosting
- Configure DNS and domains
- Implement basic security measures

## Project Requirements
Deploy a static website:
- Install Nginx or Apache
- Configure virtual hosts
- Set up custom domain (or use IP)
- Implement HTTPS with Let's Encrypt
- Configure basic access logs
- Set proper file permissions
- Optimize static content delivery

## Suggested Technologies
- Nginx or Apache
- Let's Encrypt/Certbot
- HTML/CSS/JavaScript
- Linux server (Ubuntu/Debian)

## Learning Outcomes
- Web server configuration
- SSL/TLS certificates
- DNS configuration
- File permissions and ownership

## Next Steps
After completing this project, try:
- Adding custom error pages
- Implementing HTTP/2
- Setting up caching headers
- Configuring a CDN
- Adding server-side compression
