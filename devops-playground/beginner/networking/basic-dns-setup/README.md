# Basic DNS Setup

## Overview
Configure DNS for a domain and understand DNS record types.

## Skill Level
Beginner

## Domain
Networking

## Objectives
- Understand DNS fundamentals
- Configure various DNS record types
- Set up name servers
- Troubleshoot DNS issues

## Project Requirements
Set up DNS configuration:
- Register a domain (or use a free subdomain service)
- Configure A records (IPv4)
- Configure AAAA records (IPv6)
- Set up CNAME records for subdomains
- Configure MX records for email
- Set up TXT records (SPF, DKIM)
- Verify DNS propagation
- Use DNS tools for troubleshooting

## Suggested Technologies
- DNS providers (Cloudflare, Route53, etc.)
- dig, nslookup, host commands
- Online DNS checkers

## Learning Outcomes
- DNS record types and purposes
- DNS propagation and caching
- DNS troubleshooting
- Name resolution process

## Next Steps
After completing this project, try:
- Setting up DNSSEC
- Configuring split-horizon DNS
- Implementing DNS-based load balancing
- Setting up a private DNS server
- Learning about DNS security (DoH, DoT)
