# EC2 Instance Setup

## Overview
Launch and configure an Amazon EC2 instance in AWS.

## Skill Level
Beginner

## Domain
Cloud / AWS

## Objectives
- Learn AWS fundamentals
- Launch and manage EC2 instances
- Configure security groups and networking
- Understand cloud pricing and instance types

## Project Requirements
Deploy an EC2 instance:
- Create an AWS account (Free Tier)
- Launch an EC2 instance (t2.micro)
- Configure security groups (SSH, HTTP/HTTPS)
- Set up SSH key pairs
- Connect to the instance via SSH
- Install a web server (Nginx/Apache)
- Assign an Elastic IP (optional)
- Create and attach EBS volume
- Set up basic monitoring and alerts

## Suggested Technologies
- AWS EC2
- AWS CLI
- Linux (Amazon Linux/Ubuntu)
- Nginx or Apache

## Learning Outcomes
- AWS console navigation
- EC2 instance management
- Security groups and network ACLs
- AWS pricing and cost management
- Cloud resource provisioning

## Next Steps
After completing this project, try:
- Creating AMIs for backup and replication
- Setting up Auto Scaling groups
- Implementing Load Balancers
- Using CloudWatch for monitoring
- Automating deployment with user data scripts
